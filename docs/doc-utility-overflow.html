Overflow

Overflow utilities control how content behaves when it exceeds the boundaries of its container. Use these classes to show scrollbars, hide overflowing content, or allow content to be visible outside the container. Essential for creating scrollable containers, preventing layout breaks, and managing content that exceeds fixed dimensions.

Common Use Cases
  • overflow-auto: Scrollable content areas, chat windows, code blocks
  • overflow-hidden: Image containers, card components, preventing layout breaks
  • overflow-x-scroll: Horizontal tables, image carousels, wide data grids
  • overflow-y-auto: Sidebar navigation, dropdown menus, modal content
  • overflow-visible: Tooltips, dropdowns that extend beyond parent boundaries
