Customizing with SASS/SCSS

You can use any SASS / SCSS compiler you like. If you don't like terminal or node/npm stuff, then try simpler GUI apps for Mac or Windows.
For ex. Prepros.io or Codekit app

Using only CSS

You can use final.css or final.min.css file to create any project. We also provided many starter templates to get started quickly.
