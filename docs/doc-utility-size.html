Width & Height Utilities

Width and height utilities provide a comprehensive system for controlling element dimensions using predefined size values. These utilities cover width, height, min-width, max-width, min-height, and max-height properties, allowing you to size elements consistently throughout your project. The spacing scale is based on a 4-pixel increment system, making it easy to maintain visual rhythm and alignment. Use these classes when you need to set fixed dimensions, create responsive layouts, or constrain element sizes within specific boundaries.

How to customize?

You can customize the size scale by modifying the $size_map in the scss/utilities/_sizes.scss file. This map defines all available size values that get generated into utility classes. After making your changes, recompile the SCSS to generate your customized utility classes with your preferred spacing scale.

To compile your SCSS modifications into production-ready CSS, you have two primary approaches. The first option involves using Node.js and npm tools, which is ideal if you are comfortable working in the terminal and already have Node.js installed on your system. You would install a Sass compiler package such as dart-sass or node-sass through npm, then execute the compilation command from your project directory. The second option caters to developers who prefer visual interfaces and want to avoid command-line work. Applications like Prepros or CodeKit provide intuitive graphical user interfaces where you can simply drag and drop your SCSS files to watch and compile them automatically. Both approaches produce identical CSS output, so choose whichever workflow fits your development style and comfort level.
