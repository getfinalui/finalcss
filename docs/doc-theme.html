Colors in CSS variable

All colors in the _theme.scss are converted into CSS variable. There is _variable-root.scss file which takes color map from _theme.scss and generates into CSS variable, which you can use for components or to create any custom styles. All generated color variables are stored in :root proprty in the final.css or final.min.css

Dark mode color variable

The color variables above (default light mode) will be overridden when dark mode is enabled. For example:

[data-theme=dark], .theme-dark {
  --neutral-0: #000;
  --neutral-50: #101010;
  --neutral-100: #171717;
  --neutral-200: #242424;
  --neutral-300: #373737;
  --neutral-400: #555555;
  --neutral-500: #737373;
	...
	...
}
