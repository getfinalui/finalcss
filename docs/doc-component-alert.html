Alert

Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages. You can customize alert component from the file /scss/components/_alert.scss

  • Danger or error:alert alert-danger
  • Warning: alert alert-warning
  • Success: alert alert-success
  • Informative: alert alert-info
