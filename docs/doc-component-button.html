Buttons

Finalcss comes with many button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.
Finalcss uses similar approach as per Bootstrap's btn classname. We have a base .btn class that sets up basic styles such as padding and content alignment. By default, .btn comes with a transparent border and background color.

Simple button

HTML CODE

<a href="#" class="btn">Simple button </a>

Button sizes

Use btn-lg to make buttons large, btn-sm to make smaller buttons

Small button: class="btn btn-sm
Large button: class="btn btn-lg
Icon button class="btn btn-icon
Icon button small class="btn btn-icon btn-sm
Icon button large class="btn btn-icon btn-lg

You utility classes to customize button sizes, For ex h-20 make height:80px
