Buttons

There are many button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.
Finalcss uses a style similar to Bootstrap's btn class. We have a base .btn class that sets up basic styles such as padding and content alignment. By default, .btn comes with a transparent border and background color.

Simple button

HTML CODE

<a href="#" class="btn">Simple button </a>

Default button Primary button

HTML CODE

<a href="#" class="btn btn-default">Default button </a>
<a href="#" class="btn btn-primary">Primary button </a>

Icon button

Icon buttons has rectangular shape - its width and height has same size. For that we need class name btn-icon

Icon button btn-icon
Icon button small btn-icon btn-sm
Icon button large btn-icon btn-lg

HTML CODE

<a href="#" class="btn btn-icon btn-default"> ★ </a>
<a href="#" class="btn btn-icon btn-primary"> ★ </a>

Button sizes

Use btn-lg to make buttons large, btn-sm to make smaller buttons

How to customize

You can customize by overriding css variables using inline style. Here are list of css variables

Custom pink button

HTML CODE

<button>
class="btn p-6 rounded-full text-white text-lg fw-semibold"
style="--btn-bg:#ff38a5; --btn-bg-hover:#911098;"> Custom pink button
</button>
